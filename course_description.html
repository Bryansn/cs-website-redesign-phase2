Course Descriptions

Below is a small sample of courses offered in the Computer Science program. For the complete and most updated course list, please visit the University Catalog.

Sample Courses

  • CS 1300 - Discrete Structures
    Introduction to logic, set theory, combinatorics, and formal reasoning.
  • CS 2400 - Data Structures and Advanced Programming
    Implementation and analysis of data structures such as lists, trees, and hash tables.
  • CS 3560 - Object-Oriented Design and Programming
    Software design using classes, inheritance, interfaces, and patterns.
  • CS 4310 - Operating Systems
    Overview of operating systems. Processes, threads, scheduling, memory management, and concurrency.
